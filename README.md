## Tools

### [E2E UI Test Agent](E2E%20UI%20Test%20Agent/README.md)

A Chrome/Edge extension that lets you write E2E web tests in plain language. An AI agent reads the page DOM and executes actions step by step — click, fill, navigate — and reports PASS or FAIL.

- No coding required
- Works on any website
- Supports Claude, OpenAI, Azure OpenAI, Ollama and Github Copilot
