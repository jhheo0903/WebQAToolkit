## 도구 목록

### [E2E UI Test Agent](E2E%20UI%20Test%20Agent/README.ko.md)

자연어로 E2E 웹 테스트를 작성할 수 있는 Chrome/Edge 확장 프로그램입니다. AI 에이전트가 페이지 DOM을 읽고 클릭, 입력, 이동 등의 액션을 단계별로 실행하여 PASS 또는 FAIL을 판정합니다.

- 코딩 불필요
- 모든 웹사이트에서 동작
- Claude, OpenAI, Azure OpenAI, Ollama, Github Copilot 지원
